# 📖 About

The **Student Management System** is a modern web application designed to simplify student record management. It provides an intuitive interface for storing, updating, searching, and managing student information efficiently.

This project was generated using **Google AI Studio** and enhanced with modern web technologies.

- ➕ Add Student

- ✏️ Edit Student Information
- 🗑️ Delete Student

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Arul1414/Student-Management-System.git
```

Open the project

```bash
cd Student-Management-System
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```
